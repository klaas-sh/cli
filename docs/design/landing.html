Features

Everything you need for remote sessions

klaas wraps Claude Code transparently, adding remote access capabilities without changing your workflow.

Real-time Streaming

See exactly what Claude is outputting, character by character, from any device with a browser.

Session Persistence

Close your laptop and walk away. Your session keeps running and you can reconnect anytime.

Multi-device Access

Start a task on your desktop, check progress from your phone while grabbing coffee.

Remote Input

Type prompts directly from the web interface. No need to be at your terminal.

Message History

Scroll back through terminal output to see what happened while you were away.

Secure Auth

OAuth-style device flow for CLI authentication. No passwords stored on disk.
